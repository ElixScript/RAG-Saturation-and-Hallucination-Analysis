# 🔍 RAG Saturation and Hallucination Analysis on Qwen2.5 Using IDK-MRC Dataset

Implementation and evaluation of the **Retrieval-Augmented Generation (RAG)** system to analyze the phenomena of **performance saturation** and **citation hallucination** across various numbers of retrieval documents using the **Qwen2.5-1.5B-Instruct** model.

---

## 📌 Project Description

Retrieval-Augmented Generation (RAG) is an approach that combines retrieval mechanisms with Large Language Models (LLMs) to generate answers based on relevant reference documents. Although increasing the number of retrieval documents (*K*) is expected to improve answer quality, excessive context can potentially introduce **noise**, reduce model performance, and increase the risk of **citation hallucination**.

This project implements an end-to-end RAG system using **FAISS** as the retriever and **Qwen2.5-1.5B-Instruct** as the generator. The main focus of the research is to investigate:

1. Whether answer quality continues to improve as the number of retrieval documents (*K*) increases.
2. How accurately the model references the correct documents through citation mechanisms.
3. What is the relationship between the amount of context and the occurrence of **citation hallucination** phenomena.

---

## 🎯 Objectives

* Implement an LLM-based Retrieval-Augmented Generation (RAG) system.
* Evaluate the impact of varying the number of retrieval documents on answer quality.
* Measure the accuracy rate of citations generated by the model.
* Analyze the phenomena of performance saturation and citation hallucination in the RAG system.

## 📚 Dataset

### IDK-MRC (Indonesian Knowledge Machine Reading Comprehension)

The dataset used is sourced from Hugging Face and contains a collection of contexts and question-answer pairs in Indonesian.

The dataset is used for:

* Building retrieval corpus,
* Evaluating answer quality,
* Measuring citation accuracy against ground truth documents.

sentence-transformers/paraphrase-multilingual-MiniLM-L12-v2

* Supports more than 50 languages including Indonesian.
* Embedding dimension of 384.

## 📏 Evaluation Metrics

### 1. ROUGE

Measures lexical similarity between model answers and ground truth.

### 2. BLEU

Measures answer quality based on n-gram precision.

### 3. BERTScore

Measures semantic similarity using contextual representations from multilingual BERT model.

### 4. Citation Accuracy

Percentage of cases when the model successfully references documents that actually contain the answer.

### 5. Hallucination Rate

Percentage of incorrect or irrelevant citations relative to the answers provided by the model.

## 🔍 Key Findings

* Answer quality **does not always improve** as the number of retrieval documents increases.
* Best performance was achieved at **K = 3**, which shows a balance between information sufficiency and context relevance.
* Adding documents up to **K = 10** causes a significant decrease in answer quality.
* The rate of **citation hallucination** increases drastically when the model receives too many reference documents.
* This phenomenon indicates the presence of **noise injection** effect and the possibility of **Lost in the Middle**, which is a condition when the model struggles to utilize important information located in the middle of long contexts.
